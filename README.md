# Netflix Data Analytics Project

## About This Project

I created this project to practice real-world data analytics skills using Netflix movies and TV shows data. The main goal was to explore how Netflix content has changed over time and understand trends related to genres, countries, ratings, and content growth.

This project helped me improve my skills in SQL, Python, data cleaning, and Tableau dashboard creation.

---

## Tools Used

- SQL Server Management Studio (SSMS)
- Python
- Pandas
- Jupyter Notebook
- Tableau Public

---

## Dataset

The dataset contains information about Netflix movies and TV shows, including:
- title
- type
- genre
- country
- release year
- ratings
- date added

Dataset source:
Netflix Movies and TV Shows Dataset from Kaggle

---

## What I Did

### Data Import & Cleaning
- Imported the CSV dataset using Python and Pandas
- Renamed columns for SQL compatibility
- Checked missing values
- Loaded cleaned data into SQL Server

### SQL Analysis
Used SQL queries to analyze:
- Movies vs TV Shows
- Top countries producing Netflix content
- Most common genres
- Rating distribution
- Content growth over time

### Tableau Dashboard
Built interactive Tableau charts and dashboards to visualize:
- Content growth trend
- Genre analysis
- Top countries
- Rating analysis

---

## Key Findings

- Movies make up most of Netflix’s content library
- Netflix content increased rapidly after 2015
- Dramas and International Movies are among the most popular genres
- The United States and India contribute a large amount of Netflix content
- TV-MA is one of the most common content ratings
